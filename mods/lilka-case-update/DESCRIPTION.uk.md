Оновлена версія задньої частини корпусу тепер підтримує більший акумулятор.

**Основні зміни:**
- Підтримка більшого акумулятора
- Область кнопок відрегульована для розміщення вищих кнопок
- Задня частина трохи ширша
- Додані отвори внизу кнопок для підтримки довших припаяних кнопок

**Сумісність:**
Ця версія сумісна з наявною верхньою частиною оригінального корпусу.

Базується на реміксі "Case for Lilka dev board" від Shashel.
