This is an updated case version for the Lilka Dev Board. Like the previous version, this case provides reliable board protection and convenient access to all ports and connectors.

**Main difference:**
A D-Pad cross has been added. This important change improves the gaming experience by providing more accurate and comfortable control in four-direction games.

**Installation:**
Installing the Lilka board into this case is as easy as in the previous version. Carefully place the board inside and secure it.

**UPDATE:**
After testing, it was found that the cross could press the buttons too hard, so a modified cross (D-pad Cross V2) was added.

This case is ideal for users who plan to use the Lilka Dev Board as a portable gaming console and value comfortable controls.
