# Lilka AI

An OpenAI chat app for Lilka. Supports two operating modes: on-screen chat with a virtual keyboard and a mode for interaction via the serial interface.

## Features

- Integration with the OpenAI API for response generation. API key required.
- Chat mode with an on-screen virtual keyboard or via the serial interface.
- WiFi, OpenAI API key, and model settings are configured through the web interface.
