# Guess the Number for Lilka

A simple but engaging game for Lilka where you need to guess a number from 0 to 100.

## 🎮 How to Play

- **UP/DOWN** — change the number
- **A** — confirm your guess
- **B** — exit the game

## 🎯 Gameplay

1. The screen shows a number that you choose using the UP/DOWN buttons
2. Press **A** to make your guess
3. The game will give a hint: **"Higher!"** or **"Lower!"**
4. Keep guessing until you find the right answer
5. After winning, the app will show:
   - The secret number
   - The number of attempts it took
   - An interesting fact about that number
