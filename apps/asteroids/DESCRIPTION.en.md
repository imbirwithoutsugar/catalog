# 🚀 Asteroids for Lilka

A port of the classic arcade game Asteroids (Atari, 1979). Fly through open space, shoot asteroids, and try to survive as long as possible — before one of them catches your ship.

## ✨ Features

- Inertial ship physics in the style of the original
- Asteroids spawn from the edges of the screen
- Three sound modes: silent, buzzer, speaker
- Several starting banner variants

## 🎮 Controls

| Button       | Action                       |
|--------------|------------------------------|
| UP / DOWN    | Accelerate forward / backward |
| LEFT / RIGHT | Rotate ship                  |
| A            | Shoot                        |
| SELECT       | Sound mode (in menu)         |
| START        | Start / exit                 |

## 📁 Structure

```
asteroids.lua
modules/     — code (ship, bullet, asteroid, data)
resources/   — sprites (.bmp) and sounds (.mp3)
```
