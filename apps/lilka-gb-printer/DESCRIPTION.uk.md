# Lilka Game Boy Printer

Емулятор Game Boy Printer для Lilka. Захоплює зображення з Game Boy Camera або інших ігор, сумісних з Game Boy Printer, відображає їх на екрані Lilka, зберігає на SD-карту та надає веб-інтерфейс для бездротового перегляду зображень.

## Можливості

### Прямий режим
- Захоплення даних Game Boy Printer через кабель зв'язку
- Відображення зображень на екрані Lilka в реальному часі
- Збереження зображень на SD-картку у форматі PNG

### Веб-режим
- Захоплення даних Game Boy Printer через кабель зв'язку
- Перегляд зображень через веб-інтерфейс
- Пряме завантаження зображень з браузера

## Вимоги

- **Lilka v2**
- **Game Boy** (DMG, Pocket, Color або Advance)
- **Кабель зв'язку Game Boy** (стандартний 6-контактний)
- **Перетворювач рівнів** 5V → 3.3V (4-канальний) — обов'язково!
- **Micro SD-картка** (опційно, для збереження зображень)

## Підключення

Підключіть кабель зв'язку Game Boy до роз'єму розширення Lilka через перетворювач рівнів:

```
Game Boy Pin       Перетворювач рівнів    Lilka ESP32
Pin 1 (VCC 5V)  -> HV  -> LV    -> 3.3V (Extension Port)
Pin 2 (SIN)     -> HV1 -> LV1   -> GPIO 21 (Extension Port)
Pin 3 (SCK)     -> HV3 -> LV3   -> GPIO 48 (Extension Port)
Pin 4 (SOUT)    -> HV2 -> LV2   -> GPIO 47 (Extension Port)
Pin 6 (GND)     -> GND -> GND   -> GND (Extension Port)
```

**Важливо:** Перетворювач рівнів обов'язковий! Пряме підключення пошкодить ESP32.
