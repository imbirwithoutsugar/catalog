# Dice Game for lilka.dev

A simple and engaging dice game for the [lilka.dev](https://lilka.dev/) console. Players can choose between rolling one or two dice, making it versatile for various game scenarios.

## Features

- Choose between 1 or 2 dice
- Dice rolling animation
- Sound effects for rolls and selections
- Total score tracking
- Clean and intuitive interface

## Game States

The game has three main states:
1. Welcome screen
2. Dice count selection
3. Game screen

## Controls

### Welcome Screen
- `START` - Start game

### Dice Selection Screen
- `LEFT/RIGHT` - Toggle between 1 and 2 dice
- `START` - Confirm selection and start game

### Game Screen
- `A` - Roll dice
- `B` - Return to dice selection
- `START` - Exit game
