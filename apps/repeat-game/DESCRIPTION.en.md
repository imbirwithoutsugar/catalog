# 🎮 Repeat the Sequence

**Repeat the Sequence** is a classic memory game where you must memorize and repeat a sequence of lit buttons with accompanying sounds. Each round the sequence gets longer — try to keep up with the growing challenge!

## 🕹️ How to Play

1. **Press button A** to start the game
2. **Watch** the sequence — buttons will light up and play sounds 👀
3. **Repeat the sequence** in the same order by pressing the buttons
4. If correct — the sequence gets longer! ✅
5. If wrong — game over ❌

## 🎯 Rules

- 📊 **Score**: number of correctly repeated sequences
- 🏆 **High score**: your best score ever
- 🎨 **4 buttons**: each has a unique color and sound
- ⏱️ **No time limit** — take as long as you need!

**Have fun! Good luck repeating those sequences!** 🍀✨
