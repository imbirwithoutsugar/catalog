# LoRa Transmitter - Модуль рації для Lilka

Проєкт містить схему та коди прошивок для модуля рації на ESP32 з шифруванням AES256(GCM) для плати [Lilka v2](https://github.com/and3rson/lilka). Прошивки до головної плати і модуля передавача написані на основі фреймворку [Meowui](https://github.com/Kolodieiev/meowui). Аудіо кодек — codec2.

## Можливості проєкту

- Передача зашифрованого голосу на відстань понад 250 м з антенами 5dbi
- Увімкнення/вимкнення шифрування
- Зміна каналу (в Україні дозволена для загального використання лише частота 433 мГц — канал 23)
- Генерація ключа та обмін налаштуваннями між платами Лілки

## Основні компоненти модуля рації

- Передавач — LoRa E220-400T22D (UART версія)
- МК — ESP32 WROOM на платі-адаптері
- Мікрофон — INMP441
- DC-DC понижуючий перетворювач напруги — Mini560 3.3V
- Одностороння плата 5×7 см

## Вимоги до використання

Для використання модуля рації разом з платою Лілка 2.х потрібні: комплект акумулятора, карта пам'яті для збереження налаштувань, а також комплект аудіо або підключений модуль звукової карти PCM5102A.

Живлення модуля рації окреме від плати Лілка. Комунікація з Лілкою відбувається по шині I2C за допомогою кабелю з чотирьох проводів довжиною менше 50 см.
