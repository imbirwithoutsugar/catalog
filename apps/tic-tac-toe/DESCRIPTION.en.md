# Tic-Tac-Toe for Lilka

An implementation of the classic Tic-Tac-Toe game for the Lilka platform. The player competes against an AI with basic logic and random elements. The game features win animation, pulsing highlight on the selected cell, and score display.

## Features

- **Play vs AI**: The AI has basic logic with random elements
- **Win animation**: The winning line is crossed out
- **Cell highlight**: Pulsing green frame on selected cell
- **Score**: Displays win count for each player
- **Exit confirmation**: Option to exit after confirmation
- **Lose sound**: Sad melody plays if player "O" wins
- **Win sound**: Happy melody plays if player "X" wins

## Controls

- **Arrows**: Move between cells
- **A**: Make a move / confirm exit
- **B**: Cancel exit / start new game after match ends
- **D**: Initiate exit from the game

## How to Run

Upload `tic_tac_toe_lilka.lua` to Lilka using an FTP client (e.g., Filezilla) and launch it from the Lilka menu.
