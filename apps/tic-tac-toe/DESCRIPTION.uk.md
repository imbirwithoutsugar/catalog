# Хрестики-нулики для Лілки

Реалізація класичної гри "Хрестики-нулики" для платформи Лілка. Гравець змагається проти штучного інтелекту з базовою логікою та елементами випадковості. Гра включає анімацію виграшу, пульсуюче підсвічування вибраної клітинки та відображення рахунку перемог.

## Особливості

- **Гра проти AI**: Штучний інтелект має базову логіку з елементами випадковості
- **Анімація виграшу**: Перекреслення виграшної лінії
- **Підсвічування вибраної клітинки**: Пульсуюча зелена рамка
- **Рахунок**: Відображення кількості перемог для кожного гравця
- **Підтвердження виходу**: Можливість вийти з гри після підтвердження
- **Звук при програші**: Сумна мелодія відтворюється, якщо виграв гравець "O"
- **Звук при виграші**: Радісна мелодія відтворюється, якщо виграв гравець "X"

## Керування

- **Стрілки**: Переміщення між клітинками
- **A**: Зробити хід / підтвердити вихід
- **B**: Скасувати вихід / почати нову гру після завершення
- **D**: Ініціювати вихід із гри

## Як запустити

Завантажте файл `tic_tac_toe_lilka.lua` на Лілку за допомогою FTP-клієнта (наприклад, Filezilla) та запустіть через меню Лілки.
