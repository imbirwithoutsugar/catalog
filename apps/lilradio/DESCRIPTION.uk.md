# Lilradio

Модуль FM-радіоприймача для [Лілки](https://docs.lilka.dev/)

### Компоненти

- RDA5807M FM-приймач модуль
- PAM8302 модуль аудіопідсилювача
- Динамік 1635
- SMA-роз'єм з дротом
- SMA антена
- 40-контактний кутовий роз'єм 2.54 мм
- 2x M3x5, 4x M2x5 гвинти

## Корпус

STL-файли корпусу для 3D-друку знаходяться в папці stl

## Керування

- Кнопки Right/Left — змінити частоту вгору/вниз
- Кнопки Up/Down — змінити гучність вгору/вниз
- Кнопка Select — вимкнути/увімкнути звук
- Кнопки A/D — автопошук радіостанцій вгору/вниз

Модуль радіо зберігає останню використану частоту та рівень гучності
