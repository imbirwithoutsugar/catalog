# Darkroom Timer

Таймер для проявочної кімнати (даркруму).

## Можливості

- 3 налаштовувані пресети з до 10 кроками кожен (ПРОЯВ, СТОП, ФІКСАЖ, ПРОМИВКА тощо)
- Тест Dmax — окремий секундомір для визначення часу проявки паперу
- Автоматичний перехід між кроками зі звуковими підказками
- Звукові сповіщення — початок, мітки кожні 30 секунд, зворотний відлік останніх 10 секунд, переходи між кроками
- WiFi веб-інтерфейс — редагуйте пресети та налаштування з телефону або ноутбука
- Двомовний інтерфейс — англійська / українська
- Регульована яскравість та гучність

## Керування

| Кнопка  | Функція                              |
|---------|--------------------------------------|
| UP/DOWN | Навігація                            |
| A       | Підтвердити / Старт / Продовжити     |
| B       | Назад / Пауза                        |
| L/R     | Змінити значення                     |
| START   | Пропустити крок / Редагувати назву   |
| C/D     | Додати / Видалити крок (редактор)    |
