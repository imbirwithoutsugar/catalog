# Incident LightMeter for Lilka

A basic version of an incident light meter.
Uses the BH1750 sensor and a diffuser dome from the HC-SR501 IR sensor.

Wiring:
VCC=3V3, GND=GND, pin12=scl, pin13=sda

C: Change ISO, A: Change aperture, Select: Exit
