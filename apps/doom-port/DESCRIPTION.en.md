# Doom for Lilka!

A port of the classic game Doom for Lilka on ESP32-S3.

> **Doom** is property of **id Software**. This project is an unofficial port and is not affiliated with id Software.

## Features

- Modified Doom Generic engine adapted to run on ESP32-S3
- Dynamic memory allocation to use PSRAM
- All core Doom code fits in 400 KB of main RAM
- Sound support via I2S and piezo buzzer

## Requirements

To play Doom, you additionally need the `doom.wad` file. It can be purchased or downloaded from the internet. Place the file in the root of the SD card.
