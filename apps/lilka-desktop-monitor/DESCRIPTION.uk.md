# Lilka Desktop Monitor

Стрімінг екрану комп'ютера на пристрій Lilka через WiFi. Ця прошивка забезпечує трансляцію в реальному часі на 1.69" TFT-дисплей Lilka та розроблена для безшовної інтеграції з KeiraOS.

## Вимоги

- Lilka v2 з прошитою KeiraOS
- WiFi налаштований в KeiraOS
- Обидва пристрої в одній WiFi мережі

### Крок 1: Завантаження прошивки

**Запуск з KeiraOS**
1. Зберіть прошивку за допомогою PlatformIO або завантажте готову прошивку з релізів.
2. Скопіюйте файл прошивки на SD-картку
3. У файловому менеджері KeiraOS відкрийте файл .bin
4. Пристрій перезавантажиться та завантажить прошивку
5. Автоматично підключиться, використовуючи збережені WiFi-дані Keira

### Крок 2: Запишіть IP-адресу

Після успішного підключення до WiFi на екрані відобразиться:
```
IP Address:
192.168.1.100
Waiting for connection...
```

Запишіть цю IP-адресу для наступного кроку.

### Крок 3: Встановлення Python залежностей

Клонуйте репозиторій GitHub, перейдіть до директорії `transmitter` та виконайте:

```bash
# Створіть віртуальне середовище
python3 -m venv venv

# Встановіть залежності
venv/bin/pip install -r requirements.txt
```

### Крок 4: Стрімінг екрану!

```bash
# Базове використання
venv/bin/python transmitter.py --ip 192.168.1.100

# Або активуйте venv спочатку
source venv/bin/activate
python transmitter.py --ip 192.168.1.100
```

Замініть `192.168.1.100` на IP-адресу вашого пристрою з Кроку 2.

Готово! Тепер ваш екран транслюється на дисплей Lilka.
