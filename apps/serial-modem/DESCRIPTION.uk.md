# Serial Modem

Hayes-сумісний WiFi-модем для портативної консолі Lilka ESP32-S3. Ця прошивка перетворює Лілку на вінтажний WiFi-модем, що дозволяє класичним комп'ютерам підключатися до інтернету через WiFi за допомогою стандартних AT-команд через послідовне з'єднання RS232.

Lilka підключається до комп'ютера через перетворювач рівнів RS232–TTL (MAX3232/SP3232EEN) до виводів TX та RX порту розширення.

## Основні можливості
- AT-команди — сумісність з набором команд Hayes (ATDT для набору, ATH для завершення, AT$SB для зміни швидкості тощо)
- TCP/Telnet-з'єднання — підключення до BBS та інших сервісів за допомогою hostname:port замість номера телефону
- PPP Dial-Up — підтримка протоколу PPP для виходу в інтернет. Номер для набору: 777
- Передача файлів — завантаження та вивантаження файлів на SD-картку за допомогою протоколів YMODEM/XMODEM
- Веб-інтерфейс — керування налаштуваннями модему, зміна швидкості, швидкий набір, файловий менеджер SD-картки тощо
- Дисплей — відображення стану модему на екрані Лілки, зміна налаштувань через вбудоване меню

Підтримувані швидкості: від 300 до 115200 bps. Стабільна робота: 9600 bps за замовчуванням (оптимально для PPP та звичайних з'єднань), 19200 bps рекомендовано для передачі файлів через YMODEM.
